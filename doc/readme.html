Overview

Enhanced Creatures is basically an AI mod, but it is also more than that. It is designed for Tutu and it will be compatible with BGT.

One of the main purpose of this mod is to look at each creature and see how it can be improved. By creature, I am referring to a CRE file, so it can be a monster or any adventurers (neutral or enemy). All creatures were changed one by one and many of them got new script.

It quickly became impossible to maintain, so I made a creature's editor and a script generator. I know there are already a few creature's editors but none of them have all the features I was needed, especially an export for Weidu's patching.

Script's generator was clearly needed too. Scripts quickly become big and it is a headache to correct a few bugs in a 4000 lines' script and I am not speaking about refactoring a few spells in several scripts. I have looked at SCS SSL but it didn't cover all my needs.
My generator was built with configuration and flexibility in mind. With it, scripts are about 100 times shorter (40 lines generate 4000 lines). However, it can't handle everything but it is possible to write normal blocks within scripts.

Compatibility

This mod has been designed to work exclusively with BG2: Throne of Bhaal version 26498 for several reasons:

  • It is heavily based on TobEx. It uses many of its new functionalities and bring the game to a new level. Well, without it, Enhanced Creatures would be gimped.
  • I have used ToB kit (Wild Mage) and high level abilities

While Enhanced Creatures has been developped and tested with Tutu, I highly recommend Tutu over BGT. Currently it's not compatible with BGT but will be one day.

Lastly, Spell Revisions is also required and not optional. All spellbooks were built manually with careful thinking and Spell Revisions in mind. Mage scripts are not generic but made for its own spellbook. Beside these technical reasons, think about it: without Spell Revisions, spells selection is very limited because many spells are useless. Then, most spellcasters would have a very similar spellbooks.

I don't recommend installing any others AI mod, especially SCS because its changes are huge. I know that SCS is very popular but there is a choice to make : EC or SCS, not both.

Installation

Enhanced Creatures is packaged and installed with WeiDU.

It should be installed after most mods, just before WildScreen mod.

TobEx should be installed first, but there might be some problems with Spell Revisions. It might be necessary to adapt TobEx configuration as explained in previous link.

Before installing Enhanced Creatures, it is possible to customize a few things.

  • config.ini defines many parameters for script's generation.
  • spells.csv contains all spells' informations. It is possible to change some values but any unexpected value will crash script's generator.
  • items.csv contains all items' informations, mostly potions and wands. It is possible to change some values but any unexpected value will crash script's generator.
  • all 2da files in root directory contain list of adventurers or monsters. Again, it is possible to change any value if you know what you are doing. Otherwise, it might crash at install. If you don't want to install a CRE, just delete its line

Components

Enhanced Creatures Core component (required)

Enhanced Monsters Core component (contains many subcomponents))
For all monsters, I try to make them as close as possible to P&P. Mainly sticking to 2nd edition, but taking some numbers from 3rd edition (str, dex, con, wis, int, cha).
There are quite many monsters in the game, and many variants that don't exist in P&P. In consequences, I haven't made a masterpiece because it would have taken a long time for each monster.

I highly recommand to install aTweaks P&P monsters over mine and they should be installed before Enhanced Creatures. Then, they will be detected at installation and skipped.
aVENGER and Wisp have done very high quality P&P monsters but they don't cover all monsters, so my component still have its uses.

Enhanced Adventurers Core component
There is lot to say about adventurers. Bioware has made a great game, but looking closely to each adventurers is scary! About 50-80% of numbers are wrong. It is clear that some numbers don't matter but many do. I have seen some mages with missing spells or worst, missing all spells of a specific level. I've seen fighters with axe and proficiencies in swords. Level, class, hit points, thac0 are almost never right [...]
In summary:

  • many inconsistencies
  • no kit (a few mages have a kit, but their spellbooks don't match
  • no dual wielding
  • no weapon style proficiencies
  • not always use their own magical items (not equiped)
  • no potions for level 7+
  • poor equipment for level 7+. Many high level fighters wear studded leather armor, this is why a level 5 fighter with a full plate mail (AC 1) can kill most of them in a duel.
  • very basic scripts
When rebalancing each adventurer, I have assumed than there is a full party of 6 characters. It means that some encounters might become almost impossible for solo players. Most of my tests were made with a level 5 party using basic AI scripts, and I was able to defeat most single adventurers except very high level mages.
Some high level adventurers are not meant to be killed and while they can certainly be killed with cheesy tactics, they will probably kill most parties.
Globally, with my changes, you can expect a battle matching class and level of your opponent. But bear in mind that script and items are his greatest weakness. It means that an opponent still needs more hit dices to eventually match a player.

Tweak items quantity in stores
Some wands, scrolls, or powerful arrows have an unlimited stock. It was changed to a small number (between 1 and 5).
A few temples have limited stock of potions. Party can still buy them in other temples but it is annoying and potions aren't game breaking anyway. This component gives an unlimited stock to all potions in temples.

Baldur's Gate 2 spells in stores
I have changed all caster spellbooks, they will now use all BG2 spells. It is fair to add BG2 scrolls for players.
I have added all scrolls in Sorcerous sundries and Thalantyr's stores.

Double duration for charm spells
In P&P, charm spells have a very long duration. I have doubled all durations to make them more appealing.

Issues

  • When a creature starts a battle, there might be a few seconds (between 0 and 2) before it really starts to do something. It is clearly a script issue but I wasn't able to correct it.
    If you want easy battles, attack very fast, you might drop many creatures very quickly and easily, or at least gain a subtantial advantage.
    If you want fair fight, wait until creature does something (casting, moving, or attacking).
